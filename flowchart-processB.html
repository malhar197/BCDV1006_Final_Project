In the Market

  1. Users and clients try to purchase a gun from vendor
  2. Vendor checks customer permits / Verify eligibility and request for status change
  3. Government by smart contract save the changes and commits the data
  4. Users and clients changing ownership through vendor
  5. Vendor checks customer permits and request for status change
  6. Government save the changes and commits the dataa
  7. Users and clients deliver guns for disposal to government
  8. Government takes the guns and save the changes and commits the data


Disposal

  1. Government chooses a disposal firm for gun disposal and handle the guns
  2. Government save changes and commit
  3. Disposal firm disposes the guns and requests for status change
  4. Government save changes and commit
